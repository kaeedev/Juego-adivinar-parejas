Juguemos a adivinar las parejas

Instrucciones

  • El programa preguntará cuantas filas y columnas quieres y a partir de ahí creará el tablero.
  • El juego se guardará aunque recargues la página o cierres el navegador, así que no te preocupes
  • El cronómetro se parará cuando adivines todas las parejas y podrás ver cuanto tiempo has tardado
  • Si deseas cambiar el tamaño del tablero dale click al boton "Reiniciar".
  • IMPORTANTE: Debes crear un tablero con un numero de Columnas x Filas que sea par, sino no iniciará el juego
